Perguntas Gerais

O que é o Wavera Coder?

O Wavera Coder é uma plataforma no-code baseada em inteligência artificial que permite transformar descrições em linguagem natural (prompts) em aplicações web funcionais. Nosso objetivo é democratizar o desenvolvimento de software, permitindo que pessoas sem conhecimento técnico possam criar aplicações completas.

Quem pode usar o Wavera Coder?

O Wavera Coder foi projetado para ser usado por qualquer pessoa, independentemente do nível de conhecimento técnico. É especialmente útil para:

  • Product Managers que desejam criar protótipos funcionais
  • Empreendedores que precisam validar ideias rapidamente
  • Profissionais de negócios que querem automatizar processos
  • Startups com recursos limitados para desenvolvimento
  • Desenvolvedores que desejam acelerar a criação de MVPs
Preciso saber programar para usar o Wavera Coder?

Não, você não precisa saber programar. O Wavera Coder foi projetado para permitir que pessoas sem conhecimento técnico possam criar aplicações funcionais. Basta descrever o que você deseja em linguagem natural, e nossa IA cuidará da parte técnica.

No entanto, se você tiver conhecimentos de programação, poderá aproveitar recursos avançados como personalização de código e integrações mais complexas.

Quais tipos de aplicações posso criar com o Wavera Coder?

Você pode criar uma ampla variedade de aplicações web, incluindo:

  • Sistemas de gerenciamento de conteúdo (CMS)
  • Plataformas de e-commerce
  • Sistemas de agendamento e reservas
  • Dashboards analíticos
  • Sistemas de CRM
  • Ferramentas de gestão de projetos
  • Aplicações de gestão interna
  • E muito mais

Visite nossa página de Casos de Uso para ver exemplos reais de aplicações criadas com o Wavera Coder.

Quanto tempo leva para criar uma aplicação com o Wavera Coder?

O tempo necessário para criar uma aplicação varia dependendo da complexidade do projeto. Aplicações simples podem ser criadas em minutos, enquanto projetos mais complexos podem levar algumas horas ou dias.

Em média, nossos usuários relatam uma redução de 70-90% no tempo de desenvolvimento em comparação com métodos tradicionais de programação.

Perguntas sobre o Produto

Como funciona o processo de criação de uma aplicação?

O processo de criação de uma aplicação com o Wavera Coder segue estas etapas:

  1. Descrição: Você descreve a aplicação que deseja criar em linguagem natural.
  2. Geração: Nossa IA analisa sua descrição e gera o código necessário.
  3. Visualização: Você visualiza a aplicação em tempo real e pode fazer ajustes.
  4. Refinamento: Você pode refinar a aplicação com prompts adicionais ou edição direta.
  5. Implantação: Com um clique, sua aplicação é implantada e fica disponível online.

Para uma explicação mais detalhada, visite nossa página Como Funciona.

Quais tecnologias o Wavera Coder utiliza para gerar as aplicações?

O Wavera Coder utiliza uma combinação de tecnologias modernas para gerar aplicações web completas:

  • Frontend: React, Vue.js, Angular ou HTML/CSS/JavaScript puro, dependendo das necessidades do projeto.
  • Backend: Node.js, Python (Flask/Django), Java (Spring Boot) ou PHP, conforme apropriado.
  • Banco de Dados: PostgreSQL, MySQL, MongoDB ou SQLite, dependendo dos requisitos de dados.
  • Infraestrutura: Hospedagem em nuvem com escalabilidade automática.

Nossa IA seleciona automaticamente a pilha tecnológica mais adequada para cada projeto, mas você também pode especificar suas preferências.

Posso personalizar o código gerado pelo Wavera Coder?

Sim, dependendo do seu plano. Os planos Professional, Business e Enterprise permitem acesso ao código-fonte gerado, que você pode personalizar conforme necessário.

Você pode personalizar o código de duas maneiras:

  • Através de prompts: Descreva as alterações que deseja em linguagem natural.
  • Edição direta: Acesse e edite o código-fonte diretamente no editor integrado.

Os planos Business e Enterprise também permitem exportar o código completo para hospedar em sua própria infraestrutura.

As aplicações geradas são responsivas e funcionam em dispositivos móveis?

Sim, todas as aplicações geradas pelo Wavera Coder são responsivas por padrão e funcionam bem em dispositivos móveis, tablets e desktops. Utilizamos frameworks e práticas modernas de design responsivo para garantir uma experiência consistente em todos os dispositivos.

Você também pode solicitar otimizações específicas para dispositivos móveis ou criar versões dedicadas para mobile se necessário.

Posso importar dados existentes para minha aplicação?

Sim, o Wavera Coder oferece várias opções para importar dados existentes:

  • Upload de arquivos CSV, Excel ou JSON
  • Importação de bancos de dados existentes
  • Integração com APIs externas
  • Migração de dados de outros sistemas

Nos planos Business e Enterprise, oferecemos assistência personalizada para migrações de dados complexas.

Planos e Preços

Quais planos o Wavera Coder oferece?

Oferecemos quatro planos principais para atender diferentes necessidades:

  • Starter: Ideal para empreendedores individuais e pequenos projetos.
  • Professional: Perfeito para startups e pequenas empresas.
  • Business: Ideal para empresas estabelecidas com necessidades avançadas.
  • Enterprise: Soluções personalizadas para grandes empresas.

Para detalhes completos sobre recursos e preços, visite nossa página de Planos.

Existe um período de teste gratuito?

Sim, oferecemos um período de teste gratuito de 14 dias para todos os planos. Durante o teste, você terá acesso a todos os recursos do plano Professional e poderá criar até 3 aplicações para avaliar a plataforma.

Não é necessário fornecer cartão de crédito para iniciar o teste gratuito.

Posso mudar de plano depois de assinar?

Sim, você pode fazer upgrade ou downgrade do seu plano a qualquer momento. Se fizer upgrade, a cobrança será proporcional ao tempo restante do seu ciclo de faturamento atual. Se fizer downgrade, as alterações entrarão em vigor no próximo ciclo de faturamento.

O que acontece se eu exceder os limites do meu plano?

Se você se aproximar dos limites do seu plano (como número de usuários ou armazenamento), enviaremos notificações para que você possa tomar uma decisão informada sobre fazer upgrade.

Não suspendemos serviços automaticamente quando os limites são atingidos, mas entraremos em contato para discutir as opções disponíveis.

Posso cancelar minha assinatura a qualquer momento?

Sim, você pode cancelar sua assinatura a qualquer momento. Se cancelar, você continuará tendo acesso ao serviço até o final do período de faturamento atual. Não oferecemos reembolsos para períodos parciais não utilizados, exceto quando exigido por lei.

No entanto, oferecemos uma garantia de devolução do dinheiro de 30 dias para todos os planos pagos. Se você não estiver satisfeito nos primeiros 30 dias após a assinatura, processaremos o reembolso integral.

Suporte Técnico

Como posso obter suporte técnico?

Oferecemos diferentes canais de suporte, dependendo do seu plano:

  • Starter: Suporte por email com tempo de resposta de até 48 horas úteis.
  • Professional: Suporte por email e chat com tempo de resposta de até 24 horas úteis.
  • Business: Suporte por email, chat e telefone com tempo de resposta de até 8 horas úteis.
  • Enterprise: Suporte prioritário com gerente de conta dedicado e SLA garantido.

Além disso, todos os planos têm acesso à nossa base de conhecimento, documentação e tutoriais.

Vocês oferecem treinamento para novos usuários?

Sim, oferecemos diferentes opções de treinamento:

  • Tutoriais em vídeo: Disponíveis para todos os usuários.
  • Webinars semanais: Sessões ao vivo para demonstrar recursos e responder perguntas.
  • Onboarding personalizado: Disponível para planos Business e Enterprise.
  • Treinamento corporativo: Sessões personalizadas para equipes, disponíveis para planos Enterprise.
Como são tratadas as atualizações e manutenções da plataforma?

Realizamos atualizações e manutenções regulares para melhorar a plataforma e adicionar novos recursos. Estas são geralmente programadas para períodos de baixo tráfego e anunciadas com antecedência.

Para manutenções planejadas, enviamos notificações por email com pelo menos 48 horas de antecedência. Manutenções de emergência podem ocorrer sem aviso prévio, mas são raras e tentamos minimizar o impacto.

Todas as aplicações hospedadas em nossa plataforma são automaticamente atualizadas para garantir segurança e compatibilidade.

O que acontece se eu encontrar um bug na minha aplicação?

Se você encontrar um bug em uma aplicação gerada pelo Wavera Coder, nosso suporte técnico ajudará a identificar e corrigir o problema. O processo geralmente envolve:

  1. Reportar o bug através do portal de suporte ou email.
  2. Nossa equipe analisará o problema e determinará se é um bug na plataforma ou na aplicação específica.
  3. Para bugs na plataforma, implementaremos correções que beneficiam todos os usuários.
  4. Para bugs específicos da aplicação, ajudaremos a corrigir através de prompts de refinamento ou edição direta do código.

Planos Business e Enterprise têm prioridade na resolução de bugs.

Vocês oferecem SLA (Acordo de Nível de Serviço)?

Sim, oferecemos SLAs formais para planos Business e Enterprise:

  • Business: Garantimos 99,9% de uptime e tempo de resposta de suporte de até 8 horas úteis.
  • Enterprise: Garantimos 99,99% de uptime e tempo de resposta de suporte personalizado conforme necessidades específicas.

Para planos Starter e Professional, embora não ofereçamos SLAs formais, mantemos os mesmos altos padrões de disponibilidade e suporte.

Segurança e Privacidade

Como o Wavera Coder protege meus dados?

A segurança dos seus dados é nossa prioridade. Implementamos várias camadas de proteção:

  • Criptografia: Todos os dados são criptografados em trânsito (HTTPS/TLS) e em repouso (AES-256).
  • Autenticação: Suporte a autenticação de dois fatores (2FA) e single sign-on (SSO) para planos Business e Enterprise.
  • Backups: Backups automáticos regulares com retenção configurável.
  • Monitoramento: Sistemas de detecção de intrusão e monitoramento contínuo.
  • Atualizações: Patches de segurança aplicados regularmente.
  • Isolamento: Ambientes isolados para garantir que dados de diferentes clientes permaneçam separados.

Além disso, realizamos auditorias de segurança regulares e testes de penetração para identificar e corrigir vulnerabilidades.

O Wavera Coder está em conformidade com regulamentações de privacidade?

Sim, o Wavera Coder foi projetado para estar em conformidade com as principais regulamentações de privacidade, incluindo:

  • LGPD (Lei Geral de Proteção de Dados): Conformidade com a legislação brasileira de proteção de dados.
  • GDPR (Regulamento Geral de Proteção de Dados): Conformidade com a regulamentação europeia.
  • CCPA (California Consumer Privacy Act): Conformidade com a legislação californiana.

Implementamos processos para garantir que você possa atender às solicitações de seus usuários relacionadas a direitos de privacidade, como acesso, correção e exclusão de dados.

Para clientes Enterprise, podemos fornecer documentação detalhada de conformidade e assinar acordos de processamento de dados (DPAs) conforme necessário.

Quem tem acesso aos meus dados e código?

Apenas você e os usuários que você autorizar têm acesso aos seus dados e código. Nossa equipe de suporte pode acessar seus dados apenas com sua permissão explícita para fins de resolução de problemas.

Todos os acessos são registrados e auditados. Implementamos controles de acesso baseados em função (RBAC) para garantir que apenas pessoal autorizado possa acessar informações sensíveis.

Não compartilhamos, vendemos ou alugamos seus dados para terceiros. Utilizamos seus dados apenas para fornecer e melhorar nossos serviços, conforme detalhado em nossa Política de Privacidade.

As aplicações geradas são seguras?

Sim, as aplicações geradas pelo Wavera Coder seguem as melhores práticas de segurança por padrão. Implementamos automaticamente:

  • Proteção contra injeção de SQL
  • Proteção contra Cross-Site Scripting (XSS)
  • Proteção contra Cross-Site Request Forgery (CSRF)
  • Autenticação segura com hashing de senhas
  • Validação de entrada de dados
  • Controle de acesso baseado em função
  • Cabeçalhos de segurança HTTP

Além disso, realizamos verificações de segurança automatizadas em todas as aplicações geradas e fornecemos recomendações para melhorar a segurança quando necessário.

O que acontece com meus dados se eu cancelar minha assinatura?

Se você cancelar sua assinatura, seus dados permanecerão em nossos servidores por 30 dias após o término do período de faturamento. Durante esse período, você pode reativar sua conta ou solicitar uma exportação completa dos seus dados.

Após 30 dias, seus dados serão programados para exclusão permanente. Backups contendo seus dados serão retidos por até 90 dias adicionais, após os quais serão completamente removidos de nossos sistemas.

Para planos Business e Enterprise, podemos oferecer períodos de retenção personalizados conforme necessário.

Integrações

Com quais serviços o Wavera Coder pode se integrar?

O Wavera Coder pode se integrar com uma ampla variedade de serviços e APIs, incluindo:

  • Pagamentos: Stripe, PayPal, PagSeguro, Mercado Pago
  • Autenticação: Google, Facebook, Apple, Microsoft
  • Email: SendGrid, Mailchimp, AWS SES
  • CRM: Salesforce, HubSpot, Pipedrive
  • Análise: Google Analytics, Mixpanel, Hotjar
  • Armazenamento: AWS S3, Google Cloud Storage, Azure Blob Storage
  • Mapas: Google Maps, Mapbox
  • E muitos outros

Além disso, suportamos integrações personalizadas com qualquer API RESTful ou GraphQL.

Como posso integrar minha aplicação com serviços externos?

Você pode integrar sua aplicação com serviços externos de várias maneiras:

  1. Através de prompts: Descreva a integração desejada em linguagem natural (ex: "Integrar com Stripe para processamento de pagamentos").
  2. Usando nosso catálogo de integrações: Selecione integrações pré-configuradas em nossa interface.
  3. Configuração manual: Nos planos Professional, Business e Enterprise, você pode configurar integrações manualmente fornecendo chaves de API e outros detalhes.
  4. Código personalizado: Nos planos Business e Enterprise, você pode escrever código personalizado para integrações complexas.

Nossa documentação fornece guias detalhados para as integrações mais comuns.

Posso integrar com meus sistemas internos existentes?

Sim, você pode integrar aplicações geradas pelo Wavera Coder com seus sistemas internos existentes. Isso pode ser feito de várias maneiras:

  • APIs: Se seus sistemas internos expõem APIs, você pode integrá-los diretamente.
  • Webhooks: Configure webhooks para comunicação bidirecional entre sistemas.
  • Banco de dados: Em alguns casos, é possível conectar diretamente ao seu banco de dados existente (com as devidas precauções de segurança).
  • Middleware: Para integrações mais complexas, você pode usar middleware ou serviços de integração como Zapier ou Make (Integromat).

Para planos Business e Enterprise, oferecemos consultoria personalizada para ajudar com integrações complexas com sistemas legados ou proprietários.

O Wavera Coder oferece APIs para integração com outras ferramentas?

Sim, o Wavera Coder oferece APIs RESTful e GraphQL que permitem integrar nossa plataforma com suas ferramentas e fluxos de trabalho existentes. Com nossas APIs, você pode:

  • Criar e gerenciar aplicações programaticamente
  • Automatizar processos de implantação
  • Integrar com sistemas de CI/CD
  • Acessar e manipular dados de suas aplicações
  • Gerenciar usuários e permissões

APIs completas estão disponíveis nos planos Business e Enterprise, enquanto os planos Professional têm acesso a um conjunto limitado de endpoints.

Posso exportar dados das aplicações geradas pelo Wavera Coder?

Sim, você pode exportar dados das aplicações geradas pelo Wavera Coder em vários formatos, incluindo:

  • CSV
  • Excel (XLSX)
  • JSON
  • XML
  • SQL (para backup de banco de dados)

Você também pode configurar exportações automáticas programadas para backup ou integração com outros sistemas.

Nos planos Business e Enterprise, oferecemos opções avançadas de exportação e migração de dados.
